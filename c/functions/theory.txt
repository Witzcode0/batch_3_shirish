What is function?

In C programming, a function is a self-contained block of code designed to perform a specific task. 

types of functions:

built-in function : printf, scanf, fopen, fclose
user-define function : 

syntax: 

data_type function_name(para1, para2,...paran){ // Formal para/args
    // statments of function
}

function call:
function_name(val1, val2,...valn); // Actual para/args


Example : 

#include<stdio.h>

void sum(int num1,int num2){
    printf("res: %d\n", num1 + num2);
};

void  main(){
    while(1){
        int n1, n2;
        printf("\nEnter a num1 and num2 : ");
        scanf("%d %d", &n1, &n2);
        sum(n1, n2);
    }
}
